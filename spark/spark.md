- 1.特性简介
```
1).Speed
  Run workloads 100x faster than Hadoop
2).Ease of Use
  Write(编写) applications quickly in 
    Java, Scala, Python, R, and SQL.
3).Generality(概述) 
  Combine SQL, streaming, and complex analytics.
  Spark powers a stack of libraries including 
    - Spark SQL and DataFrames,
    - Spark Streaming,
    - MLlib for machine learning,
    - GraphX.
4).Runs Everywhere
  Spark runs on Hadoop, Apache Mesos, Kubernetes, standalone, or in the cloud. 
  It can access diverse(多种) data sources.
  You can run Spark using its 
    standalone cluster mode, 
    on Amazon EC2, 
    on Apache Hadoop YARN, 
    on Apache Mesos,
    or on Google Kubernetes. 
  Access data in
    Apache Hadoop HDFS, 
    Apache Cassandra, 
    Apache HBase, 
    Apache Hive, 
  and hundreds of other data sources.
```

- 2.Spark SQL
```
Spark SQL is Apache Spark's module for working with structured data.
加工结构化数据
```

- 3.Spark Streaming
```
Spark Streaming makes it easy to build scalable fault-tolerant streaming applications.
可以很容易得构建可扩展、可容错的流式应用
```

- 4.Spark MLlib
```
MLlib is Apache Spark's scalable machine learning library.
可扩展的机器学习库
```

- 5.Spark GraphX
```
GraphX is Apache Spark's API for graphs and graph-parallel computation.
一个图形和图平行计算的API
```
