#### Zookeeper
```
https://zookeeper.apache.org/
Apache顶级项目

Apache ZooKeeper enables highly reliable distributed coordination.
ZooKeeper可以实现高度可靠的分布式协调(协作)

ZooKeeper is a centralized service for maintaining 
  configuration information, 
  naming, 
  providing distributed synchronization, 
  and providing group services.

Because Coordinating Distributed Systems is a Zoo.

ZooKeeper is a high-performance coordination service for distributed applications.

You can use it off-the-shelf(现成的) to implement 
  consensus(共识), 
  group management, 
  leader election(选举), 
  and presence(存在) protocols.
```
